UNREAL ENGINE PROJECTS

Arena Shooter

Language used:- C++, visual scripting

This was the game I created while learning unreal engine. This game was the final result of the course i was learning from UDEMY.This game have third person shooting with RPG-like enemies which attacks Player when close enough or if triggered by the player shooting. In this player have to kill all the enemies without dying. Enemies health, size and speed increases with closing distance from winning, which increases dificulty level for the player.

Roll Ball

Language used:- visual scripting

This was the game I created which was the first game i made without a character as a player and wanted to learn about controlling pawn as a player character.In this game player have to reach from one point to another without Dying and collecting reward points. In this there are animated obstacles which helps or kills the Player.

Survival Horror

Language used:- visual scripting
